#### 2. 두 실수를 키보드에서 읽고 메모리에 저장한 후 더한 후 화면에 출력하는 C프로그램을 작성하시오.

```
#include <stdio.h>
int main() {
	float a, b;
	scanf_s("%f%f", &a, &b);
	printf("%f\n", a + b);
}
```
```
#include <stdio.h>
int main() {
	double a, b;
	scanf_s("%lf%lf", &a, &b);
	printf("%lf\n", a + b);
}
```
```
3.4
4.5
7.900000
```
