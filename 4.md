#### 4. 두 실수를 메모리에 저장하고 함수 Add()에서 더한 후 리턴한 값을 화면체 출력하는 C언어 프로그램을 작성하시오.

```
#include <stdio.h>
float Add(float a, float b) {
	return a+b;
}

int main() {
	printf("%.2f\n", Add(3.3, 4.4));
}
```
```
#include <stdio.h>
double Add(double a, double b) {
	return a + b;
}

int main() {
	printf("%.2lf\n", Add(3.3, 4.4));
	return 0;
}
```

7.70
